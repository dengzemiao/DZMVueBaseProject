# DZMVueBaseProject

node: v16.14.0

自用 Vue 空项目框架，只配置常用的项目使用环境

#### 一、项目情况

- 默认 `无 CSS 预处理器`、`无 JS 语法检测`

- 支持 `Vuex 数据持久化`、`Axios 基础请求配置`、`其他常用基础配置`、`各种文件下载`

- 如果拷贝到新的工程下，注意隐藏文件是否拷贝成功。
